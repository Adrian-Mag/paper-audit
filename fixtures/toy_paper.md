# Effects of Paced Breathing on Working Memory Task Performance: A Pilot Study

**Abstract**

We investigated whether a brief paced-breathing exercise affects performance on a working memory task. Twenty-four undergraduate students completed a computerized digit-span task under two within-subject conditions: a five-minute paced-breathing exercise performed at a fixed rate of six breaths per minute, and a five-minute silent-reading control. Mean response time on the digit-span task decreased by 18% following the paced-breathing condition relative to the control condition (612 ms vs. 746 ms), a difference that was statistically significant (t(23) = 3.41, p < .01). Accuracy did not differ significantly between conditions.

## 1. Introduction

Working memory performance is known to fluctuate with short-term changes in physiological arousal. Slow, paced breathing is widely used as a relaxation technique, and prior laboratory work has linked slowed breathing rate to reduced physiological arousal during simple attention tasks. Whether this translates into measurable changes on a working memory task specifically is less clear, since arousal and working memory performance are not always related in a simple or consistent way across tasks.

The present pilot study tested whether a brief paced-breathing exercise, performed immediately before a working memory task, changes performance on that task relative to a matched control activity. We focused on a single, well-characterized working memory measure, the digit-span task, so that any observed effect could be interpreted against a specific, narrow behavioral baseline rather than a broad and heterogeneous battery of tasks.

## 2. Method

### 2.1 Participants

Twenty-four undergraduate students (mean age 20.3 years, 15 women and 9 men) at a single university volunteered for course credit. All participants reported no history of cardiovascular or respiratory conditions.

### 2.2 Procedure

Each participant completed a computerized digit-span task twice: once immediately following a five-minute paced-breathing exercise performed at a fixed rate of six breaths per minute, and once immediately following a five-minute silent-reading control activity. Condition order was counterbalanced across participants, and the two sessions were separated by a ten-minute break. The digit-span task required participants to recall increasingly long sequences of digits in order; response time and accuracy were recorded for each trial.

## 3. Results

Mean response time on the digit-span task was 612 ms (SD = 88 ms) following the paced-breathing condition and 746 ms (SD = 101 ms) following the control condition. This 18% reduction was statistically significant (t(23) = 3.41, p < .01, Cohen's d = 0.61). Accuracy on the digit-span task did not differ significantly between the paced-breathing condition (94.2%) and the control condition (93.6%), t(23) = 0.52, p = .61.

## 4. Discussion

The present pilot study found that a brief paced-breathing exercise was followed by faster response times on a digit-span task relative to a matched control activity, with no accompanying change in accuracy. This pattern is consistent with a shift in processing speed on this specific task rather than a demonstrated change in memory capacity more broadly.

We note two limitations. First, our sample was limited to undergraduate students at a single university, and the present design cannot establish whether the effect would be observed in other age groups or populations. Second, the digit-span task captures only one narrow aspect of cognitive function, and we did not test other task types, so the generality of the effect across cognitive domains is untested here.

Taken together, these findings show that paced breathing improves cognitive performance and represents a promising general intervention for enhancing mental function in everyday settings.

## 5. Conclusion

This pilot study provides evidence that a brief paced-breathing exercise can speed responses on a working memory task under controlled laboratory conditions. Paced breathing improves cognitive performance and may offer a simple, low-cost tool worth pursuing in future work.
